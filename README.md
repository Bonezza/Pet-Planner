# Pet Planner v4 🐾

## New in this version

- Add real pet photos from iPhone
- Choose a different accent colour for every pet
- Includes Original Purple and Heliotrope
- Edit existing pet profiles
- Upgraded pet cards
- Bottom navigation tabs
- Habitat Build Mode preview
- Smooth theme changes when switching pets
- Quick feeding log per pet
- Existing v3 profiles are migrated automatically

## Upload to GitHub

Replace these files in your repository:

- `index.html`
- `style.css`
- `app.js`
- `manifest.json`
- `README.md`

GitHub Pages may take a minute or two to refresh.
