# Squirtle Tracker 🐢

A tiny black-and-purple feeding tracker for Squirtle.

## Features

- Log when Squirtle is fed
- See the latest feeding time
- Count logged meals
- Undo the latest feeding
- Saves data in the browser

## Run it

Open `index.html` in a browser, or publish the folder with GitHub Pages.
